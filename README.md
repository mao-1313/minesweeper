# 課題9：マインスイーパー

`tcell` を使ったターミナル上で動くマインスイーパーを実装する。

---

## 完成イメージ

```
+---+---+---+---+---+---+---+---+---+
|   |   |   |   |   |   |   |   |   |
+---+---+---+---+---+---+---+---+---+
|   | 1 | 1 | 1 |   |   |   |   |   |
+---+---+---+---+---+---+---+---+---+
|   | 1 | * | 1 |   |   |   |   |   |
+---+---+---+---+---+---+---+---+---+
|   | 1 | 1 | 1 |   |   |   |   |   |
+---+---+---+---+---+---+---+---+---+

[矢印キー] 移動  [Space] 開く  [f] フラグ  [q] 終了
```

---

## ゲームルール

- 9×9 のマス、地雷10個
- マスを開けたとき地雷なら **ゲームオーバー**
- 数字は周囲8マスの地雷数を示す
- 数字が0のマスを開けると周囲を**再帰的に自動展開**
- 地雷以外の全マスを開けたら **クリア**
- `f` キーでフラグを立てて地雷候補をマーク

---

## 学習ステップ

### Step 1 - ボードの定義と初期化

盤面の状態を構造体で表現し、地雷をランダムに配置する。

```go
type Cell struct {
    IsMine    bool
    IsOpen    bool
    IsFlagged bool
    Adjacent  int  // 周囲の地雷数
}

type Board struct {
    Cells  [9][9]Cell
    Cursor [2]int  // [row, col]
}

func NewBoard() *Board          // 地雷をランダム配置
func (b *Board) calcAdjacent()  // 全セルの周囲地雷数を計算
```

### Step 2 - 再帰的なマス展開

数字が0のマスを開けたとき、周囲を自動的に展開する。

```go
func (b *Board) Open(row, col int) GameState
```

- 地雷なら `GameOver`
- 再帰で隣接する空マスを展開
- 全マス開いたら `GameClear`

### Step 3 - tcell で画面描画

`tcell` を使ってボードをターミナルに描画する。

```go
func drawBoard(s tcell.Screen, b *Board)
func drawCell(s tcell.Screen, row, col int, cell Cell, isCursor bool)
```

### Step 4 - キー入力とゲームループ

キー入力を受け取ってカーソル移動・マス展開・フラグ操作を行う。

```go
// キー操作
// ↑↓←→ : カーソル移動
// Space  : マスを開く
// f      : フラグのトグル
// q      : 終了
```

### Step 5 - ゲームオーバー・クリア画面

結果に応じてメッセージを表示し、`r` キーでリスタートできるようにする。
